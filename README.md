# 6weeks-of-code

6 week programming

Rules:
1. Learns what Need to
2. Code every day for 6 weeks ( work on single project)

RULES:
1. The Nopy Rule - do not copy past code. If needed a code then write it own.
2. The Noogle Rule - if you stuck do not google instead try everything you think, no matter. If you still stuck for 3 days then google it.
3. The Nooral Rule - do not use neural networks (AI) eg, Chat GPT, GEMINI, ETC. You use you get dumber, you unlearned the skills.
